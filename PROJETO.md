# Site do Super Mario Brothers

Projeto desenvolvido para praticar conceitos básicos de desenvolvimento web.

## 📌 Descrição
Este projeto foi criado com o objetivo de praticar HTML, CSS e JavaScript, aplicando conceitos vistos em aula e em estudos pessoais.

## ⚙️ Funcionalidades
- Estrutura de página com HTML
- Estilização com CSS
- Interações simples com JavaScript (botões, eventos, validações básicas)

## 🛠️ Tecnologias Utilizadas
- HTML
- CSS
- JavaScript

## 🎯 Objetivo do Projeto
Projeto acadêmico / estudo pessoal voltado para aprendizado e composição de portfólio para estágio em desenvolvimento web.

Projeto concluído (podendo receber melhorias futuras).
